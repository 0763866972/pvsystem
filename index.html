Phân tích tính toán chi tiết

1. Công suất tịnh tại PCC (Pnet)

P_net = P_pv - P_load = 0 - 5000 = W
Q_net = 0 - 500 = -500 VAr

2. Độ biến thiên áp trên dây (ΔV)

ΔV ≈ (P_net × R + Q_net × X) / V_nguon
ΔV = ( × 0.5 + -500 × 0.1) / 230 = V

3. Điện áp đo được tại PCC (Vpcc)

V_pcc = V_nguon + ΔV = 230 + () = V
